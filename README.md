# DeckHeads

A virtual deck of cards you can play with. I made this to help me prototype some diy card games. Maybe you'll find it useful !

1. Touch/Mouse to grab a card.
2. Move to drag around.
3. Double tap to show/hide a card.
4. Tap/click once to rotate card.
5. Touch/Mouse drag on background to create selection box and select multiple cards.
6. Drag to move multiple selected card.
7. Shake to shuffle selected cards.
